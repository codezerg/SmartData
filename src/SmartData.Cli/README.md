# SmartData.Cli

`sd` — command-line tool for managing SmartData servers, databases, tables, columns, indexes, data, import/export, backups, and stored procedures.

## Install

```bash
dotnet tool install --global SmartData.Cli \
    --add-source https://smartdata-apis.netlify.app/nuget/v3/index.json
```

## Usage

```bash
sd --help
```

See the [docs](https://smartdata-apis.netlify.app/tools/cli/) for full command reference.
